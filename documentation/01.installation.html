--- title: Installation ---

Compatibility

The download contains the version that is compatible with both the EE3 and EE4. Older versions are available on Github: EE2 and EE1.

Installation

- Download and unzip the latest version of Low Random. - Copy the low_random folder to your /system/user/addons directory. - In your Control Panel, go to the Add-on Manager and install Low Random. - All set!
